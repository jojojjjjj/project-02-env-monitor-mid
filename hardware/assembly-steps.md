# 组装步骤 | Assembly Steps

> 项目 02 · 温湿度网络时钟 · 从零到桌面摆件的分阶段组装
> Project 02 · Stage-by-stage assembly from parts to desktop ornament
>
> 两种搭法共用同一套元器件（见 `BOM.md`），区别只在承载方式：**方案 A 面包板 MVP**（零焊接，默认）与 **方案 B PCB 版**（焊接打样，进阶）。
>
> 循序渐进原则：**先电源 → 再 MCU → 再传感器/显示 → 再 WiFi 模块**。每接一类测一类，分而治之，出问题能立刻定位。

## 方案 A · 面包板 MVP 组装（默认 / Default）

**目标**：零焊接，Day 1–8 在面包板上跑通全部功能。

### 准备：面包板电源轨

1. 将 STM32L433 开发板插入面包板**中央**，横跨中间凹槽，USB 口朝外便于插线。
2. 用杜邦线把开发板的 **3V3** 引到面包板**红色电源轨**（+），**GND** 引到**蓝色电源轨**（−）。
3. **上电自检**：插 USB，用万用表测红色轨↔蓝色轨 = **3.30V ± 0.1V**。不对则立刻拔 USB 检查。

> 为什么先搞电源轨：面包板上所有器件都从这两条轨取电。轨没整对，后面全是雷。

### 阶段 1：电源（已随上一步完成）✅

### 阶段 2：MCU 自检（Day 2）

1. 开发板只接 USB，不接任何外设。
2. 烧录「点灯 + printf」例程（课程 `software/src/` 提供）。
3. 打开串口助手（115200），应看到 `printf` 输出的启动信息；板载 LED 闪烁。
4. **不通过则停**：检查 USB 数据线、串口驱动、波特率——见 `troubleshooting.md`。

### 阶段 3：SHT31 温湿度传感器（Day 3）

1. SHT31 模块 4 针：VCC→3V3 轨、GND→GND 轨、SCL→**PB13**、SDA→**PB14**。
2. 烧录 SHT31 读取例程，串口应每秒打印：`T=25.3C H=56.7%`。
3. 用手捂传感器，温湿度应上升。

### 阶段 4：SSD1306 OLED 显示（Day 4）

1. OLED 4 针：VCC→3V3、GND→GND、SCL→**PB8**、SDA→**PB9**。
2. 烧录 OLED 例程，屏幕显示温湿度。
3. 注意 OLED 与 SHT31 **不在同一组引脚**（PB8/PB9 vs PB13/PB14），不要接错。

### 阶段 5：蜂鸣器 + 按键（Day 8）

1. 蜂鸣器 IN→**PA1**、GND→GND（3 脚版还有 VCC→3V3）。烧 PWM 例程，应发声。
2. 按键 KEY1：一脚→**PC13**，另一脚→GND；KEY2：一脚→**PA15**，另一脚→GND。烧 GPIO 例程，按键触发 LED 翻转。

### 阶段 6：ESP-01S WiFi 模块（Day 5，最后接）⚠️

1. **先单独测 AT**（推荐用 USB-TTL 串口模块直连 ESP-01S，见 `troubleshooting.md` 的「ESP8266 AT 无响应」一节）。确认 AT 固件响应正常后，再接到 STM32。
2. ESP-01S 接线：
   - VCC → 3V3（**禁 5V**）
   - GND → GND
   - CH_PD(EN) → 3V3（必须拉高）
   - TX → **PB11**（MCU RX）
   - RX → **PB10**（MCU TX）
3. 在 VCC 与 GND 间并 **100μF 电解 + 0.1μF 陶瓷**电容（ESP 瞬态电流大，防重启）。
4. 烧录 WiFi+SNTP 例程，串口应看到 `AT` → `OK` → `WIFI CONNECTED` → NTP 时间。

### 阶段 7：整合（Day 6–7）

1. 全部外设就位后，烧录完整时钟固件。
2. OLED 应显示：**时间（网络授时）+ 温度 + 湿度**。
3. 每小时蜂鸣器报时；按键可切换显示/调时。

## 方案 B · PCB 版组装（进阶 / Advanced）

**目标**：Day 9–10 把验证好的电路做成焊接成品桌面摆件。需基础焊接能力。

### B-1：画 PCB（Day 9 上午）

1. 用 **立创EDA** 或 **KiCad** 新建工程，按 `wiring-guide.md` 的引脚表连线。
2. 关键布局建议：
   - STM32L433 放中央，用**排母插座**（不直接焊芯片，防烫坏、可拔插替换）。
   - ESP-01S 用 2×4 排母竖插。
   - OLED 用 4 针排针，便于拔插。
   - SHT31 用排针，远离 MCU 热源（MCU 会发热影响测温精度）。
   - USB 口、电源滤波电容放板边。
   - 100μF+0.1μF 滤波电容紧贴 ESP-01S 的 VCC 引脚。
3. 走线原则：电源线粗（20mil+）、信号线细（10mil）、I2C 上拉电阻（4.7kΩ × 2，SHT31 那条硬件 I2C2 总线）。
4. 导出 Gerber，上传**嘉立创**打样（5 片，约 <!-- ¥5 -->，2–3 天到货）。

### B-2：焊接（Day 9 下午）

> **新手焊接四原则**：①先矮后高（电阻→排针→排母→插件）②烙铁温度 320–350°C ③焊点光亮圆滑、不虚焊不连锡 ④焊完逐点用万用表测通断。

1. **先焊低矮元件**：电阻、电容、排针。
2. **再焊排母**：STM32 插座、ESP-01S 插座、OLED 排针。
3. **最后焊接口**：USB（若板载）、电源接线端子。
4. 焊完**清洗**（酒精 + 棉签擦松香渣）。

### B-3：通电自检（焊接后必做）

1. **先不插芯片**。用万用表测 PCB 上 3V3 轨↔GND 轨**电阻**：应 > 1kΩ（不能短路）。
2. 插 USB，测 3V3 轨电压 = 3.30V。不对立即拔电查短路。
3. 确认电源 OK 后，**断电**，再插 STM32、ESP-01S、OLED、SHT31。
4. 上电，按方案 A 的阶段 2→7 顺序逐项验证。

### B-4：外壳（Day 9–10，可选）

1. 3D 打印或亚克力激光切割外壳（参考尺寸见下）。
2. OLED 开窗 30×20mm；USB 开窗 15×10mm；SHT31 侧面留通风孔（测温准）。
3. M3 铜柱+螺丝固定 PCB 与外壳。

## 通用：新手焊接避坑 | Beginner Soldering Tips

- **烙铁沾锡前先上松香**：助焊剂让焊点亮、不虚焊。
- **焊点呈圆锥状、表面光亮**就是好焊点；发暗、豆腐渣状是「冷焊」（虚焊），要重焊。
- **连锡**用吸锡带吸掉，或用吸锡器。
- **芯片怕烫**：STM32/ESP 一律用排母插座，绝不直接焊到板上。
- **极性**：电解电容、LED、蜂鸣器有正负极，焊反会爆/不响。电解电容负极是白条一侧。
- **通风**：焊接时开窗或开排风扇，松香烟雾有害。

## 质量检查清单 | Quality Checklist

**电气检查：**
- [ ] 3V3↔GND 无短路（万用表蜂鸣档不响）
- [ ] 上电后 3V3 轨 = 3.30V ± 0.1V
- [ ] 所有 GND 连通（共地）
- [ ]（PCB 版）每条走线导通、无虚焊

**功能检查：**
- [ ] 串口 printf 能输出（USART1）
- [ ] SHT31 读到温湿度（I2C2）
- [ ] OLED 显示正常（PB8/PB9 软件 I2C）
- [ ] 蜂鸣器响（PA1 PWM）
- [ ] 按键触发中断（PC13/PA15）
- [ ] ESP-01S AT 响应（USART3）
- [ ] WiFi 联网 + NTP 授时成功
- [ ] OLED 显示时间+温湿度

**外观检查（PCB 版）：**
- [ ] 焊点光亮无虚焊
- [ ] 元件无歪斜
- [ ] 外壳缝隙均匀
